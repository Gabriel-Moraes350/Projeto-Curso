# Projeto-Curso
Projeto do meu curso de desenvolvimento web front-end, onde tento aplicar na prática o que está sendo aprendido, e botando a mão na massa pra valer dessa vez.
Foi um dos primeiros sites que eu (tentei)desenvolver, não está sendo aplicado nenhuma metodologia especial em relação ao css, foi um projeto onde pude aprender na prática como fazer certar funções em jquery, e conseguir corrigir erros e o mais importante, aprender!
O sistema de rotas também não está nada adequado, nada automático, não foi usado nenhum sistema de back-end, o importante era aprender mesmo.
Para funcionar, deverá ter alterado no seu git clone os arquivos 'scroll.js' e 'functions.js' para suas pastas na variável directory.
